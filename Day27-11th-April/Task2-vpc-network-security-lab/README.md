# VPC Lab Files

This package contains the scripts used in the VPC + Network Security practical:

- `validate-lab.sh` — validation script for the lab
- `cleanup.sh` — cleanup script to remove resources

## Runtime files created during the lab
These are generated while you complete the practical:

- `vpc-id.txt`
- `public-subnet-id.txt`
- `private-subnet-id.txt`
- `igw-id.txt`
- `public-rt-id.txt`
- `my-ip.txt`
- `bastion-sg-id.txt`
- `private-sg-id.txt`
- `vpc-lab-key-2.pem`
- `bastion-id.txt`
- `private-id.txt`

## Notes
- The lab uses the `us-east-1` region.
- The private connection was tested through a bastion host.
