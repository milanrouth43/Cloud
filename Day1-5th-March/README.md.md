# Linux Bash Scripting Practice (Tasks 1–22)

## Overview
This repository documents my Linux Bash scripting practice.  
I completed 22 small scripting tasks using the Linux terminal to learn automation, file management, and basic system administration.

Since these tasks were performed as hands-on practice, the results are documented through screenshots instead of saved script files.

The goal of this practice was to improve my understanding of:

- Bash scripting fundamentals
- Linux terminal commands
- File and directory management
- System monitoring
- Basic automation
- Git operations

--------------------------------------------------

## Environment Used

Operating System: Ubuntu Linux  
Shell: Bash  
Editor: Nano  
Tools Used:
- Git
- Linux Terminal
- Basic Linux utilities (ls, grep, awk, find, df, free, etc.)

--------------------------------------------------

## Tasks Completed

### Task 1 – Print Name and Date
A script was written to ask the user for their name and print a greeting with the current date.

### Task 2 – System Information
Displayed system hostname, uptime, and kernel version.

### Task 3 – Create Multiple Files
Used a loop to create multiple text files automatically.

### Task 4 – Add Content to Files
Added sample text content inside the created files.

### Task 5 – User Greeting Script
Created an interactive script to greet the user.

### Task 6 – Mathematical Operations
Performed multiplication and division using user input numbers.

### Task 7 – File Existence Check
Checked whether a specific file exists in the directory.

### Task 8 – Disk Usage Monitor
Displayed disk usage and printed a warning if usage exceeded a limit.

### Task 9 – Backup Script
Created a backup directory and copied text files into it.

### Task 10 – Top CPU Processes
Displayed the top processes consuming CPU resources.

### Task 11 – Count Files
Counted the number of files in the current directory.

### Task 12 – Count Files in Folder
Counted files inside a user-specified directory.

### Task 13 – List Filenames
Looped through the directory and printed each file name.

### Task 14 – List System Users
Displayed system users by reading the `/etc/passwd` file.

### Task 15 – Error Log Finder
Searched system logs for the keyword "error".

### Task 16 – Install Package
Installed Git using the Linux package manager.

### Task 17 – System Statistics
Displayed hostname, disk usage, memory usage, and system uptime.

### Task 18 – Dated Backup Folder
Created a backup folder named with the current date.

### Task 19 – Delete Old Files
Used the `find` command to remove files older than a specific time.

### Task 20 – Memory Usage Calculation
Calculated RAM usage percentage using Linux commands.

### Task 21 – Git Repository Checker
A script was created to check whether a project folder exists.  
If it exists, it pulls the latest changes.  
If it does not exist, it clones the repository from GitHub.

### Task 22 – Git Project Updater
Created a script that navigates to the project folder and pulls the latest code from the repository.
