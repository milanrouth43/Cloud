🧰 1. Install Terraform
✅ Windows (LATEST – recommended)
winget install HashiCorp.Terraform

✔ This is a valid modern method using Windows Package Manager

✅ Linux (Ubuntu/Debian – OFFICIAL HashiCorp method)
sudo apt-get update && sudo apt-get install -y gnupg software-properties-common
wget -O- https://apt.releases.hashicorp.com/gpg | gpg --dearmor | sudo tee /usr/share/keyrings/hashicorp-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update
sudo apt-get install terraform

✔ This is the official installation flow from HashiCorp

✅ Verify
terraform -version
☁️ 2. Install AWS CLI
✅ Windows (LATEST official way)

👉 AWS recommends MSI installer (not package managers)

Download & install:

https://awscli.amazonaws.com/AWSCLIV2.msi

✔ This is the official supported method for Windows

✅ Linux (latest official command)
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install

✔ Direct installer ensures latest version

✅ Verify
aws --version
🔐 Configure
aws configure
☁️ 3. Install Azure CLI
✅ Windows (LATEST – official)
winget install Microsoft.AzureCLI

✔ Microsoft officially supports WinGet install

✅ Linux (Ubuntu/Debian – official)
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

✔ This is Microsoft’s official install script

✅ Verify
az version
🔐 Login
az login
